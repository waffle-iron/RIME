What is RIME?

RIME is a chat application, based on the RINE app from Steins;Gate 0. Nothing too fancy.

It is being made in Ruby, using the Ruby on Rails framework, and uses Travis CI for Continuous Integration and running tests etc. The app will be hosted on Heroku, with the repo being publically available on my GitHub, if you wish to clone it.

For more information about myself, and my other projects, please do visit my blog and portfolio site.

Planned Features

  • The Chat itself; instant text messages (think Facebook messenger etc.)
  • Stickers and images in messages.
  • User signup and Login.
